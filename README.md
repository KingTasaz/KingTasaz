## Baloker
- Languages: `C 🔷`
- Status: In Progress
Baloker is a combination of standard Texas Hold'em Poker with the popular game [Balatro](https://store.steampowered.com/app/2379780/Balatro/). This is my first big step into the world of C programming, and I am very proud of how far I've come. Baloker uses the SDL3 graphics library for a clean and reactive UI, and connects to a custom server software running on my Raspberry Pi allowing for live multiplayer and user accounts.

## Unbloomed
- Langauges: `Python 🐍` `GLSL 🖥️`
- Status: (Very much) In Progress
Unbloomed is a Metroidvania Platformer that I am currently developing with a few of my friends. I wont share many details because a lot of it is supposed to be a secret, and I wont share many images because the current art is very very temporary. What I would like to share is that Unbloomed is written almost entirely in Python! Many people like to say that Python is a slow language, but I am pushing the language to its limits, using numpy and openGL to directly interface with the GPU and maximize performance.

## Stock Ticker
- Languages `Python 🐍`
- Status: Finished
Stock Ticker was originally a board game that we played in my grade 12 ICS class. One of our assignments was to recreate the game digitally. Our instructions were not very detailled and only required that the gameplay loop was implemented, and there were no requirements for it being visually pleasing. I, of course, can't stand when my UI doesnt look nice. I used pygame a my graphics library, and had to figure out how to add nice background blur, glow on buttons when the cursor is near, and animated text box inputs. The game features many settings to customize gameplay however you'd like.
